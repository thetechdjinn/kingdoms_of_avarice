Game Settings Editor

Configure all game settings. Changes take effect immediately after saving.

Max Characters Per Player
max_characters_per_player
Default number of characters each player can create. Can be overridden per-player in the User Editor.
IP Access Mode
ip_access_mode
Controls how IP access rules are applied. In blocklist mode, all IPs are allowed unless explicitly blocked. In allowlist mode, all IPs are blocked unless explicitly allowed.
Character Auto-Save Interval
character_save_interval_ms
How often connected players' vitals (HP, mana) are automatically saved to the database in milliseconds. Lower values provide more protection against data loss from crashes but increase database load. Valid range: 10000-600000ms (10 seconds to 10 minutes). Default: 60000ms (60 seconds)
Base Energy
combat_base_energy
Base energy pool available per combat round before level multipliers.
Default Weapon Speed
combat_default_weapon_speed
Energy cost for weapons without a specified attack speed. Lower values mean faster attacks.
Max Attacks Per Round
combat_max_attacks_per_round
Maximum number of attacks any combatant can make per round, regardless of remaining energy.
Round Interval (ms)
combat_round_interval_ms
Time between combat rounds in milliseconds.
Unarmed Speed
combat_unarmed_speed
Energy cost for unarmed attacks (punching, kicking).
Combat Level Energy Multipliers
combat_level_multipliers
Energy multiplier per combat level (not character level). Higher multipliers give more attacks per round.
Combat Level Multiplier
Combat Level Accuracy Bonus
combat_level_accuracy_bonus
Accuracy bonus percentage per combat level (not character level).
Combat Level Bonus %
Runic Currency Name
currency_runic_name
The display name for the highest tier currency. Default is "runic". This name is used in inventory displays and commands.
Copper Coins per Encumbrance Point
currency_copper_per_enc
How many copper coins equal 1 encumbrance point. Default: 25
Silver Coins per Encumbrance Point
currency_silver_per_enc
How many silver coins equal 1 encumbrance point. Default: 25
Gold Coins per Encumbrance Point
currency_gold_per_enc
How many gold coins equal 1 encumbrance point. Default: 15
Platinum Coins per Encumbrance Point
currency_platinum_per_enc
How many platinum coins equal 1 encumbrance point. Default: 10
Runic Coins per Encumbrance Point
currency_runic_per_enc
How many runic coins equal 1 encumbrance point. Default: 4
Training Base Cost
training_base_cost
Base cost in copper to train from level 1 to level 2. Higher levels cost exponentially more based on the multiplier. Default: 28
Training Cost Multiplier
training_cost_multiplier
Exponential multiplier applied per level. E.g., at 1.8x, level 3 costs 1.8x the base, level 4 costs 3.24x (1.8^2). Default: 1.8
Initial Character Points
initial_character_points
Number of Character Points (CP) given to new characters for allocating stats. Default: 100
